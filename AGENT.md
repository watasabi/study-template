# Agent guidelines — rust_study

Este repositório é um **estudo** (notas Obsidian + apenas notas).

## Objetivo

Ajudar a organizar aprendizado: capítulos em `chapters/`, dados em `data/`,
links em `02-Links-uteis.md`, visão geral em `00-Dashboard.md`.

## Convenções

- Tags YAML: `estudo/rust-study`, `tema/rust`.
- Wikilinks compatíveis com Obsidian (`[[nota]]`).
- Linha Python: 79 caracteres; `uv run ruff check` / `ruff format` se existir
  `pyproject.toml`.

## O que não fazer

- Não commitar segredos em notas ou em `data/raw/`.
- Não apagar a estrutura de tags sem alinhar com [[VAULT-SYNC]].
